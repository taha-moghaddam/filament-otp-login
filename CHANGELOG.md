# Changelog

All notable changes to `filament-otp-login` will be documented in this file.

## 1.0.0 - 2025-02-01

- Initial release
- OTP login for Filament v5
- Mobile in URL (no session for pending state)
- Configurable request block and same-OTP resend limit
- Pluggable OTP sender (interface)
- English and Persian translations
